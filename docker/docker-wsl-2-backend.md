# WSL 2

## O que significa?

Nada mais é que um "Windowns Subsystem for Linux", ou seja é um subsistema Linux rodando dentro do nosso windows

### Como faz para instalar wsl no windows? 

```bash 
wsl --status
```

### Como instalar caso eu não tenha instalado?

explore a documentação: [documentação para instalar wsl](https://learn.microsoft.com/en-us/windows/wsl/install)

Ou você pode executar o seu terminal como administrador e rodar o comando

```bash
wsl --install
```
